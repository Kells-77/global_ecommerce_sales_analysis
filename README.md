# global_ecommerce_sales_analysis
Analysis of a public global e-commerce sales data set

[View the dashboard (PDF)](global_ecommerce_sales_analysis.pdf)

I used a public dataset found on Kaggle to explore global e‑commerce sales and profit from 2023–2025. The dashboard looks at performance by region, country, and payment method, and includes filters for year and payment type. My goal was to build a clear, easy‑to‑read Power BI report that highlights basic trends and makes it simple to compare different parts of the world.
